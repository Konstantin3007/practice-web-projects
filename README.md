# Учебные web-проекты по производственной практике

Репозиторий создан в рамках производственной
(технологической, проектно-технологической) практики
в Университете «Синергия».

## Проект 1. Web-приложение «Блог»
Учебный проект для отработки навыков web-разработки:
- регистрация и авторизация пользователей
- создание и редактирование постов
- подписки на пользователей
- комментарии
- сортировка записей по тегам

## Проект 2. Web-версия книжного магазина
Учебный проект с пользовательским и административным интерфейсами:
- каталог книг
- сортировка по категориям, авторам и годам
- покупка и аренда книг
- управление списком книг

## Проект 3. Web-приложение «Дневник путешествий»
Учебный проект для ведения дневника путешествий:
- пользовательские записи
- просмотр путешествий других пользователей
- изображения, геолокация и стоимость поездки
